# Autou Frontend

Case Prático AutoU - Luan Rodrigues Carlos

## Como Iniciar

1. Instale as dependências:

```bash
npm install
```

2. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

3. Acesse a aplicação em [http://localhost:3000](http://localhost:3000)

## Tecnologias Utilizadas

- Next.js
- React
- TypeScript
